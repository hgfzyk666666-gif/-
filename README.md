## 🌟 核心功能与技术亮点

### 1. 🤖 AI 智能对话记账 (Gemini 驱动)
* **多模态交互**：支持语音或文本对话记录收支，自动解析时间、金额、分类及备注。
* **智能预算与分析**：支持通过对话设定预算、查询历史账单，并提供深度的收支分析建议。

### 2. 📊 模块化动态图表（基于 MPAndroidChart）
* **顶部灵活筛选**：支持按周/月/年动态切换时间跨度，伴有动态周期文本提示（如“2026年06月”）。
* **资产信息概览**：首部高亮展示黄色背景的**收入、支出与净结余**核心数据卡片。
* **支出环形图 (PieChart)**：采用中心留空高精设计，居中显示当前周期总支出，外圈优雅呈现各分类比例。
* **收支趋势折线图 (LineChart)**：双轨排布收入（黄）与支出（红）的走势，支持手势缩放、水平滑动与触点详情查看。
* **高花销排行柱状图 (HorizontalBarChart)**：直观横向排列消费类型，大额花销一目了然。

### 3. 📄 结构化财务报表分享（基于 iText7）
* **一键安全导出**：图表页面右上角集成系统原生 "Share" 标识导出按钮。
* **本地沙盒生成**：实时基于筛选数据，在本地安全生成包含**周期汇总、收支明细分类表格、逐笔明细清单**的结构化 PDF。
* **原生无缝分享**：生成后自动唤起 Android 原生 `ACTION_SEND` 意图，支持直接发送至微信、保存至本地文件或调用系统“截图及打印”处理。

### 4. 🗺️ 优雅的交互导航
* **底部导航栏**：基于 `NavigationBar` 组件，实现“记账 (Chat)”页面与新增“图表 (Charts)”页面间的随心、流畅切换。
